## Installation

Use the package manager [npm](https://nodejs.org/en/download/) to install.

```bash
npm i
```

Split terminal

```bash
cd client
npm i
```

\_Create a new .env file from .env.example and fill your key

## Run

```bash
npm run nor
```

Split terminal

```bash
cd client
npm start
```
